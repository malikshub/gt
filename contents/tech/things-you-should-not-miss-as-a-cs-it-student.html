Things You Should Not Miss As A CS/IT Student

by Shubham MalikMay 2nd, 2021

Get Free Resources

For a CS/IT student, there are many more things to learn apart from the curriculum to boost your career. Various things could have helped me even more if I could have known about those things earlier in my career still "Better Late Than Never".

In this post, I will be talking about various resources which you could avail yourself of for free if you're a student, and if are not then I'll also be talking about various alternatives that are already available for free.

GitHub Student Developer Pack

GitHub Student Developer Pack is the only thing you need to learn to code, it provides the student with various developer tools in one place so that you can learn by doing. The pack includes various paid software as well as learning courses which you can access for absolutely free.

To apply for this pack up must be 13 years old, a student and you should have a GitHub account. Although they ask for a school-issued email id if you don't have one, then you can simply use any dated document that states you are a student for example you can take a letter from your school/university on their letter-pad that has a date on it and says that you are a student.

Top resources I have been using from this Pack

Canva: I use canva to create graphics for my website, it is really useful and easy t use, this pack includes 12 months premium subscription of Canva.
Alternatively, you can use Freepik.com to get free templates and you can edit those templates using Photopea.com which is a free alternative to Adobe Photoshop.

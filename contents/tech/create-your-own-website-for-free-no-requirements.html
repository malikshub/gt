Create Your Own Blogging Website For Free

by Shubham MalikMay 10th, 2021

Getting Started

If you know how to copy and paste stuff then you can create your own website, and if you also know the shortcut commands for both the operations then it is even better. Jokes apart, with the number of resources available today it's really easy to create and publish a website, using drag and drop tools you can create the site you want within few hours or even minutes.

In this post, I'll be discussing how I created my own blogging website for absolutely free and help you create your own even if you don't know anything related to website creation and deployment. Before starting I recommend getting few resources that we would use GitHub Student Developer Pack, Bootstrap Studio, VS Code.
The GitHub Student Developer Pack is required to get Bootstrap Studio for free with other valuable resources or you can claim your student license for Bootstrap Studio here .

Designing Website Using Bootstrap Studio

Bootstrap Studio is a powerful tool that can be used to create a website with a drag and drop feature. Bootstrap Studio has a variety of templates and components that can be added and edited into your site easily. I will be using a pre-built template from Bootstrap Studio for the purpose of this tutorial.
Alternative to Bootstrap Studio is Bubble, Webflow, and WordPress. For the deployment of the website, we can use the deployment service provided by either Github or Bootstrap Studio itself.

  1. After the installation of Bootstrap Studio, Create a new project.

  2. Select a template to begin, or select a black page.

  3. Fill in the required details, and hit create.

  4. The template I selected can be edited now.

  5. Some Basic Tools

    6. Most componets of the website can be directly edited just by clicking on them.
    On the left is a panel which has various pre built components such as header, footer, forms, and a lot more. you can drag and drop these components to your page directly.
    On the top right are two important button one for the live preview of the website and the other for the deployment of the website, which I will be showing below.

    For a detailed tutorial prefer the official docs by Bootstraphere. You can also learn some basic HTML and CSS within few minutes from w3schools.

Deploying Website Using Bootstrap Studio

  1. Create a Website

    2. Click the Publish button, and create a website by selecting a type, choose from a custom domian or select the free domain with the subdomain as shown below.

  2. Publish the Website

    3. Go to setings and then publish settings, and select the domain that you created or use your custom domain.

  3. Generated Output

Deploying Website Using GitHub

Create a GitHub account if you don't have one, then create a repository that will contain all the contents of your website. For the deployment of the site go to the repository settings, then go to pages.

  1. First export your website

    2. From the bar below the the title bar, select the export option and click in export after selecting a generic location. You can choose to minify files that will increase the page loading speed.

  2. Create a GitHub Repository

    3. After you create a repository, uplaod the exported files to the repository.

  3. Go to Settings

  4. Generated Output
